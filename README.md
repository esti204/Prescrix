Prescrix

Prescrix is a mobile and web-based application designed to help users securely store, manage, and organize medical prescriptions, doctor information, and visit history in one place.

Overview

In many cases, patients lose paper prescriptions or forget important medical advice given by doctors. Prescrix solves this problem by providing a centralized digital platform where all medical records can be accessed anytime.

Features
Prescription Management
Upload prescriptions as images or PDF files
Organize prescriptions by date and doctor
Secure cloud storage
Doctor Management
Add and manage doctor details
Store phone number, email, address, and specialization
Quick access to saved doctors
Visit Notes
Record doctor advice and diagnosis
Add personal notes for each visit
Keep track of treatment history
Medical Timeline
View complete history of doctor visits and prescriptions
Organized in a chronological timeline
Search Functionality
Search prescriptions, doctors, and notes
Quickly find past records
Reminders
Medicine reminders
Appointment reminders
Family Profiles (Planned)
Manage medical records for multiple family members

Run the app

Motivation

This project was built to solve a real-world problem faced by many people — losing prescriptions and not having a proper record of medical history. The goal is to create a simple, reliable, and accessible solution.
